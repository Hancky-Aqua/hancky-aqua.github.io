# 曝光

# 曝光三标准

  • 主题元素正确曝光
  • 场景观感还原
  • 暗部不死黑，亮部不死白（不损失细节）

# 曝光三要素

  • 光圈 A
  • 快门 S
  • 感光度 ISO

# 互易率

  • 光圈和快门之间存在互补关系，每 1 档对应 2 级亮度。
补充说明

快门与光圈是相互配合、互相补充的关系，也就是常使用的互易率。但是，当光线强度异常而曝光时间极短或极长时，这一定律会失效，使结果难以预测，因而有 “互易率失效” 这一术语。如果要保持曝光量不变，光圈开大一级，速度就要提高一级；光圈缩小一级，速度就要放慢一级。例如：用光圈 11，速度 1/60 秒，当光圈改用 8 时，速度就要改成 1/125 秒。主题要求表现景物一定的景深时，应优先考虑光圈大小。如果被摄对象背景杂乱，就选用大光圈，可以虚化背景；如拍摄全景适用小光圈，这样有足够的景深展现出全貌。拍摄运动物体，要求表现动感和速度时，应先考虑快门速度，再配以合适的光圈。
快门速度大于 1 秒或小于 1/1000 秒，互易率会失效！（来源：百度百科 - 互易率）

# 曝光补偿 (EV)

  • 白加黑减原则：拍摄整体偏亮（白）的场景时要增加曝光，拍摄整体偏暗（黑）的场景时要减少曝光。
    • 目的：防止相机自动测光后颜色失真（如把白雪拍成灰雪）。

# 向右曝光

  • 对数码相机而言可以适当向右曝光，以保留更多细节，后期再进行调整。
    • 直方图辅助实时取景
    • EV

# 光圈优先 (A/AV) 模式的拍摄流程

确认 光圈 --> 确认 安全快门 （焦距倒数）--> 根据快门调整 ISO --> 调整 EV

# 大光比场景

# 判断

  • 肉眼判断
    • 常见情况：日出、日落、舞台
  • ✔️直方图
    • 两头凸，中间凹

# 应对方法

  • 原则：平衡光比≠没有光比

  • 风光场景

    • 曝光合成（包围曝光、AED、HDR）
    • 宁欠勿过，后期提亮，保留高光

  • 人物场景

    • 使用外部光源补光（闪光灯、LED 等）
    • 拍摄剪影

# 对焦

# 对焦方法

  • ✔️拍哪对哪（最常用）
  • 锁定对焦
  • 陷阱对焦

# 虚焦原因 (AF)

  • 超过最近对焦距离
  • 未打开自动对焦 (AF)
  • 对焦和快门呈分离状态
  • 场景太暗，没有反差，相机无法自动识别

# 背景虚化的拍法

  • 背景离主体远
  • 相机离主体近
  • 大光圈 <2
  • 长焦（尼康 800 定）

# 光线

# 光线的属性

  • 强度
  • 色温
  • 白平衡
    • 通常使用自动白平衡 (AWB)，后期调整
  • 方向
    • 通常侧光体现立体感
  • 软硬
    • 影响明暗交界的过渡平缓程度
  • 自然光与人造光
    • 自然光（太阳光）：柔光
    • 人造光
      • 持续光源：功率低，营造氛围感
      • 瞬间光源：功率高，不易控制曝光

# 常见布光

# 三点布光

  • 主光
  • 辅助光
  • 背景光（发丝光）

# 伦勃朗光

  • 常用于男性肖像
  • 特点：脸部倒三角
  • 主光角度：高位斜下 45°

# 蝴蝶光（派拉蒙光）

  • 常用于女性肖像
  • 特点：鼻下阴影呈蝴蝶状、脸颊阴影瘦脸效果
  • 主光角度：正上方斜下
    • 可在下方配反光板补光

# 噪点控制

# 噪点类型

  • 明度噪点
  • 色彩噪点

# 控制方法

  • 前期：提升进光量
    • 人造光
    • 环境光
    • 向右曝光
      • 慢门
      • 大光圈
      • 高 ISO
  • 后期：去噪点软件 (DeNoiseAI)
  • 前后期结合可以做到高 ISO 高画质

# 反光控制

  • 利用反光
    • 玻璃人像
    • 倒影风光
  • 消除反光
    • 偏振镜
    • 拍摄时镜头贴玻璃
    • 用深色布料包裹住镜头

# 如何拍好身边的美

  • 寻找光线
  • 寻找细节
  • 多角度拍摄
  • 长期拍摄同一主题
  • 摄影，不止摄影

  • 明确的主题

  • 简洁的画面
  • 好看的形式

  • 细节完备
