# Real Number Space

# Real Number System

  • 定义实数系统的两种方法
方法1 自然数→整数→有理数→实数

Approach 1. Start with the set of natural numbers N\mathbb{N}N, construct the set Z\mathbb{Z}Z of all integers (N∪{0}∪(−N))(\mathbb{N}\cup\{0\}\cup(-\mathbb{N}))(N∪{0}∪(−N)), and next, the set Q\mathbb{Q}Q of rationals and then the set of R\mathbb{R}R of real numbers either as the set of all Cauchy sequences of rationals or as Dedekind cuts. The step going from Q\mathbb{Q}Q to R\mathbb{R}R vis Cauchy sequences is also available for completing any incomplete metric space.

方法2 域(field)定义：满足结构(algebraic)公理、序(ordering)公理、完备性(completeness)公理

Approach 2. Define the set of real numbers, R\mathbb{R}R, as a set that satisfies three set of axioms. The first set is algebraic involving addtion and multiplication. The second set is on ordering that, with the first, makes R\mathbb{R}R an ordered field (Royden 1988). The third set is a single axiom known as the completeness axiom. Thus R\mathbb{R}R is defined as complete ordered field.

  • 此处采用方法 2 进行讨论

Monoid（幺半群）

A set S{S}S equipped with a binary operation S×S→S{S}\times{S} \to{S}S×S→S, which we will denote ⋅\cdot⋅, is a monoid if it satisfies the following two axioms:
Associativity
For all aaa, bbb and ccc in S{S}S, the equation (a⋅b)⋅c=a⋅(b⋅c)(a \cdot b) \cdot c = a \cdot (b \cdot c)(a⋅b)⋅c=a⋅(b⋅c) holds.
Identity element
There exists an element eee in S{S}S such that for every element aaa in S{S}S, the equations e⋅a=ae \cdot a = ae⋅a=a and a⋅e=aa \cdot e = aa⋅e=a hold.
In other words, a monoid is a semigroup with an identity element. It can also be thought of as a magma with associativity and identity. The identity element of a monoid is unique. For this reason the identity is regarded as a constant, i.e.\mathrm{i.e.}i.e. 0-ary (or nullary) operation. The monoid therefore is characterized by specification of the triple (S,⋅,e)({S}, \cdot , e)(S,⋅,e).
A monoid in which each element has an inverse is a group. (From Wikipedia: Monoid)

# Bounded Monotone Sequence

  • 有界 (bounded) 序列：A sequence is bounded if the set of terms from the sequence is bounded.
  • 单调 (monotone) 序列：A sequence {xn:n∈N}⊆R\{x_n:n\in\mathbb{N}\}\subseteq\mathbb{R}{xn​:n∈N}⊆R is
    • (1) increasing if xn≤xn+1,∀n∈Nx_n\leq x_{n+1},\forall n\in\mathbb{N}xn​≤xn+1​,∀n∈N;
    • (2) decreasing if xn≥xn+1,∀n∈Nx_n\geq x_{n+1},\forall n\in\mathbb{N}xn​≥xn+1​,∀n∈N;
    • (3) strictly increasing if xn>xn+1,∀n∈Nx_n>x_{n+1},\forall n\in\mathbb{N}xn​>xn+1​,∀n∈N;
    • (4) strictly decreasing if xn<xn+1,∀n∈Nx_n<x_{n+1},\forall n\in\mathbb{N}xn​<xn+1​,∀n∈N;
  • A sequence is monotone if it is either increasing of decreasing.

# Theorem on Limit of Bounded Monotone Sequence
